# Pragenx AI

Pragenx AI is a proactive personal assistant designed to remind, predict, and protect you from life's everyday stress. It monitors your lifestyle, health, and work to provide peace of mind through intelligent automation and privacy-first design.

## Key Features

- **Proactive Intelligence**: Anticipates your needs before you even ask.
- **Privacy & Trust**: Your data remains under your control with a privacy-first architecture.
- **Minimalistic Design**: A clean, modern aesthetic featuring transparent SVG iconography.
- **24/7 Monitoring**: Continuous oversight of your bills, health, travel, and work.
- **Seamless Integration**: Works across your digital life to reduce cognitive load.

## Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- A modern web browser

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/pragenxai-glitch/Pragenx-project.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Pragenx-project
   ```

### Development
To run the project locally:
```bash
npx serve .
```
The site will be available at `http://localhost:3000`.
